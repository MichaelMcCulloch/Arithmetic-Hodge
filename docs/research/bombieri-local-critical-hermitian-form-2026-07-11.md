# Polarized Bombieri local critical Hermitian form

Date: 2026-07-11

Status: strict-compiling proof discovery. The diagonal local critical formula
has been polarized into an actual algebraic sesquilinear form on smooth
`BombieriTest`; productionization and the later extension to Yoshida's
non-smooth low modes remain.

## Why this layer is needed

The existing endpoint theorem
`bombieriFunctional_quadratic_eq_local_critical_form_le_two` gives only a
diagonal expression for a single test `g`:

```text
2 Re(Mg(1) conjugate(Mg(0)))
  + (1/(2 pi)) integral K(v) |Mg(1/2+iv)|^2 dv,

K(v) = Re psi(1/4+iv/2) - log pi.
```

Yoshida's Riesz corrections and corrected Gram matrix need cross pairings,
not only diagonal values. Treating the displayed quadratic formula as if it
were already a bundled Hermitian form would leave linearity, conjugation, and
cross-integrability unproved.

## Cross-integrability

The polarized critical integrand is

```text
K(v) * conjugate(Mf(1/2+iv)) * Mg(1/2+iv).
```

The existing theorem `bombieriCriticalArchKernel_integrable g` proves that
`K(v) Mg(1/2+iv)` is integrable. The probe bounds `Mf` by the uniform norm of
the Fourier transform of its critical logarithmic Schwartz pullback, then
uses `Integrable.bdd_mul`. Thus the cross integral is proved Bochner
integrable; it is not justified only by formal polarization.

The probe separately proves additivity and scalar behavior of the integral in
both variables. The first variable carries the conjugate scalar, matching
Mathlib's conjugate-linear-first convention.

## Bundled form

The pairing is

```text
B(f,g) = conjugate(Mf(1)) Mg(0)
       + conjugate(Mf(0)) Mg(1)
       + (1/(2 pi)) integral K(v)
           conjugate(Mf(1/2+iv)) Mg(1/2+iv) dv.
```

It is bundled as

```lean
def bombieriLocalCriticalForm :
  BombieriTest →ₗ⋆[ℂ] BombieriTest →ₗ[ℂ] ℂ
```

The kernel proves

```text
star (B(g,f)) = B(f,g).
```

For the integral term this uses `integral_conj`, the fact that `K(v)` is
real, and the already proved cross-integrability. No symmetry axiom is added.

## Diagonal and endpoint bridge

On the diagonal,

```text
conjugate(M1) M0 + conjugate(M0) M1
  = 2 Re(M1 conjugate(M0)),

conjugate(Mv) Mv = normSq(Mv).
```

The probe proves these identities and then exports

```lean
theorem bombieriFunctional_quadratic_eq_bombieriLocalCriticalForm_le_two
    (g : BombieriTest) ...
    (hsupport : tsupport g ⊆ Icc a b)
    (hratio : b / a ≤ 2) :
  bombieriFunctional (bombieriQuadraticTest g) =
    bombieriLocalCriticalForm g g
```

This is an exact composition with the existing prime-free endpoint theorem.
It proves a representation identity, not positivity.

## Honest scope boundary

`BombieriTest` consists of smooth compactly supported functions on the
positive half-line. Yoshida's cutoff sine/cosine low modes are naturally
Lebesgue `L²` vectors and are not themselves bundled `BombieriTest` values.
Therefore this form is the correct polarized smooth-test layer, but it is not
yet by itself the ambient form required by
`externalLowRieszCorrectionOfSqBound` for those low modes.

The later construction must do one of the following, with proofs:

1. extend `B` to an algebraic ambient space generated by the smooth tail and
   finitely many external Yoshida modes using the source pairing formulas; or
2. define each low-tail functional directly from the coefficient/pairing
   formula and prove that it agrees with `B` on smooth approximants.

Either route must retain the raw low-mode Gram entries, the cross pairings,
and the tail form in one conjugation convention. It must not claim that the
form is bounded on all ordinary circle `L²`.
