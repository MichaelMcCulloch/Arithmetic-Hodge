## Primary-source correspondence

The construction follows Hiroyuki Yoshida, *On Hermitian Forms attached to
Zeta Functions*, sections 3, 4, and 6:

- DOI: <https://doi.org/10.2969/aspm/02110281>
- PDF: <https://projecteuclid.org/download/pdf_1/euclid.aspm/1534359132>

The correspondence is:

1. Lemma 3 supplies positivity and `L²` coercivity on a sufficiently high
   algebraic Fourier tail.
2. Section 4 equips that tail with `sqrt((v,v))`, completes it, and extends
   bounded low-mode pairings.
3. Proposition 3 applies Riesz representation in that form completion.
4. Equations (6.11)--(6.13) and (6.21)--(6.22) turn squared pairing bounds into
   squared correction-norm bounds.

## Algebraic form and isolated wrapper

The algebraic input is:

```lean
structure PositiveHermitianForm
    (V : Type*) [AddCommGroup V] [Module ℂ V] where
  form : V →ₗ⋆[ℂ] V →ₗ[ℂ] ℂ
  conj_apply : ∀ x y, star (form y x) = form x y
  re_apply_self_nonneg : ∀ x, 0 ≤ (form x x).re
  definite : ∀ x, form x x = 0 → x = 0
```

The form-norm carrier is a genuine wrapper rather than a reducible type
synonym:

```lean
structure FormSpace (q : PositiveHermitianForm V) where
  of :: toV : V
```

Only the algebraic additive-group and complex-module structures are
transported from `V`. An `InnerProductSpace.Core` with
`inner x y = q.form x.toV y.toV` then generates the new
`NormedAddCommGroup` and `InnerProductSpace` instances. This prevents an
ambient `L²` topology from leaking through typeclass inference.

The exact basic API is:

```lean
FormSpace.inner_eq_form
FormSpace.norm_sq_eq_form_re

abbrev FormSpace.Completion (q : PositiveHermitianForm V) :=
  UniformSpace.Completion (FormSpace q)

FormSpace.completion_inner_coe
FormSpace.completion_norm_coe
```

Thus the completed norm literally extends Yoshida's
`‖v‖_form = sqrt((v,v))`.

## Bounded pairing and exact squared budget

For an algebraic functional `ell : V →ₗ[ℂ] ℂ`, the probe constructs:

```lean
FormSpace.boundedFunctional
FormSpace.completedFunctional
FormSpace.rieszCorrection
FormSpace.rieszCorrection_inner_coe
FormSpace.norm_rieszCorrection_le
```

The source-shaped squared bound is preserved exactly:

```lean
theorem FormSpace.pairing_bound_of_sq_bound
    {S : ℝ} (hS : 0 ≤ S)
    (h : ∀ x : FormSpace q,
      ‖ell x.toV‖ ^ 2 ≤ S * ‖x‖ ^ 2) :
    ‖ell x.toV‖ ≤ Real.sqrt S * ‖x‖

noncomputable def FormSpace.rieszCorrectionOfSqBound :
  FormSpace.Completion q

theorem FormSpace.norm_sq_rieszCorrectionOfSqBound_le :
  ‖FormSpace.rieszCorrectionOfSqBound ell hS h‖ ^ 2 ≤ S
```

If Yoshida's analytic calculation supplies `S_source / mu`, that whole
quotient is passed as `S`; the conclusion is the required
`‖v_i‖_form² ≤ S_source / mu`.

## External low-mode interface

The concrete tail lives in an algebraic subspace `V`, while a low mode
`w` belongs to a larger algebraic ambient space `A`. The true downstream
interface therefore takes an ambient Hermitian form and an inclusion:

```lean
noncomputable def FormSpace.externalLowRieszCorrectionOfSqBound
    (B : A →ₗ⋆[ℂ] A →ₗ[ℂ] ℂ)
    (hB : ∀ x y, star (B y x) = B x y)
    (incl : V →ₗ[ℂ] A) (w : A) {S : ℝ} (hS : 0 ≤ S)
    (h : ∀ x : FormSpace q,
      ‖B (incl x.toV) w‖ ^ 2 ≤ S * ‖x‖ ^ 2) :
    FormSpace.Completion q
```

It proves both orientations and the budget:

```lean
FormSpace.coe_inner_externalLowRieszCorrection
FormSpace.externalLowRieszCorrection_inner_coe
FormSpace.norm_sq_externalLowRieszCorrection_le
```

This handles Mathlib's convention correctly: the inner product is
conjugate-linear in the first slot, and Hermitian symmetry converts Yoshida's
printed orientation without changing the absolute-value estimate.

## Coercive map to the ambient completion

For an already complete ambient normed space `U`, the source-shaped bound

```lean
mu * ‖x.toV‖ ^ 2 ≤ (p.form x.toV x.toV).re
```

gives:

```lean
FormSpace.coercive_underlying_bound_of_form

noncomputable def FormSpace.coerciveCompletionToUnderlyingOfForm :
  FormSpace.Completion p →L[ℂ] U

FormSpace.coerciveCompletionToUnderlyingOfForm_coe

theorem FormSpace.norm_coerciveCompletionToUnderlyingOfForm_le :
  ‖FormSpace.coerciveCompletionToUnderlyingOfForm hmu hcoercive‖ ≤
    1 / Real.sqrt mu
```

This is the continuous canonical map mentioned after Yoshida Proposition 3.
The inequality points from the form norm to the ambient norm; it does not
assert that the unbounded form is continuous on `L²`.

## Deliberate caveat

Coercivity alone makes the completed map to ambient `L²` continuous, but
does not by itself prove that the completed map is injective. Yoshida's
Proposition 4 supplies an additional separation/closability argument.

The generic production module must not claim injectivity. It is not needed to
construct the Riesz correction for an algebraic tail: the canonical map from
`FormSpace q` into its own completion is already isometric and injective.
If a later theorem needs to identify arbitrary completed vectors inside
`L²`, Proposition 4 must be formalized as a separate analytic result.
